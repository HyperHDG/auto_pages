Detailed Description

This file provides the function lapack_solve.

This is a wrapper file to provide LAPACK based functions that have the ability to solve (dense) local systems of linear equations in an efficient way. The functions daxpy_ and dnrm2_ are needed to provide the three functions that solve local systems of equations. From these three functions, one is chosen to be used in the remainder of the code (i.e., dgesv_ cf. LAPACK manual for further details).
