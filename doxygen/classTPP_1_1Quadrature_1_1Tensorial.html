Detailed Description

template<typename quadrature_t, typename shape_t, typename return_t = double>
class TPP::Quadrature::Tensorial< quadrature_t, shape_t, return_t >

General integrator class on tensorial hypergraphs.

Template Parameters
quadrature_tThe one-dimensional quadrature rule applied.
Parameters
shape_tType of shape functions.
Template Parameters
return_tFloating type specification. Default is double.
Authors
Andreas Rupp, Heidelberg University, 2021.
